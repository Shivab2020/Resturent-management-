# SmartRestro UI Demo

Tagline: “A modern restaurant management UI for India’s tier-2 businesses.”

This single-page demo showcases how a smart restaurant management system solves common issues in tier‑2 city restaurants. It focuses on UI/UX and flow — no backend or database required; everything runs from static data and local state.

## Modules

- Dashboard: key stats and mock charts (Daily Sales, Profit Trend, Best‑selling Dish)
- Orders & Billing: order table, add order modal, mark ready, invoice modal
- Menu: editable grid, add item, availability toggle with color badges
- Inventory: quantity and status indicators with “Restock” toast
- Feedback: comments with emoji sentiment, summary bar, AI Insight popup (mock)
- Staff & Attendance: list with present/absent toggle, add staff modal
- CRM: customer cards with visits/points/favorite dish, filter Top Customers, Send Offer (mock)
- Online Order + Reservation: form with confirmation card and WhatsApp notify (mock)

## Tech Stack

- React + Vite
- Tailwind CSS (Poppins + Inter fonts)
- Radix UI (Dialog & Switch)
- Recharts (Line/Area/Bar/Pie charts)
- Lucide React (icons)
- Sonner (toasts)

## Getting Started

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open the local URL shown in the terminal (e.g., `http://localhost:5173/` or alternate port)

## Design

- Colors: Warm orange `#FFA500` + white + grey accents
- Responsive: Mobile and tablet friendly
- Dark mode: Toggle in the navbar
